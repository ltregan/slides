ğŸ”’ Install K8S Credentials

The application runtime requires connectivity to an OCI Tenancy. A credential file for the development environment is provided

Check kubectl context: 
kubectl config current-context
# docker-for-desktop
Check kubectl namespace configuration: 
kubectl config view | grep namespace
Download K8S secrets.yaml and install: 
kubectl create -f secrets.yaml
