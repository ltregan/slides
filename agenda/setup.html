🔒 Install Credentials

The application runtime requires connectivity to an OCI Tenancy. A credential file for the development environment is provided.

  1. Download K8S secrets.yaml
  2. Install Secrets
kubectl create -f path/to/secrets.yaml
Next: Start

Run

  1. Change directory to mushop-oci-demo
  2. Deploy in K8S
kubectl apply -f deploy/kubernetes/mushop.yaml
