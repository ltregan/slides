    > kubeconfig.yaml
    This is an access configuration for an OKE cluster to be used in cases where you are unable to provision, or use a local cluster
    Installation:
    1. Backup existing kube/config: 
      cp $HOME/.kube/config $HOME/.kube/config.bak
    2. Merge downloaded kubeconfig.yaml: 
      KUBECONFIG=$HOME/.kube/config:$HOME/Downloads/kubeconfig.yaml \
                    kubectl config view --flatten > $HOME/.kube/config
      ⚠️ NOTE: Path to Downloads/kubeconfig.yaml and change as necessary
    3. Verify the mushop-workshop context: 
      kubectl config get-contexts
