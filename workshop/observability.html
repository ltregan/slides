``` helm status mushop-setup ``` TL;DR Find the section on Grafana. It will contain instructions on connecting to the Grafana instance running on the K8s cluster. Get the admin password: 
kubectl get secret -n mushop-setup mushop-setup-grafana \
  -o jsonpath="{.data.admin-password}" | base64 --decode ; echo
Proxy from your machine to the grafana service running in Kubernetes: 
kubectl port-forward -n mushop-setup \
  svc/mushop-setup-grafana 3000:80
