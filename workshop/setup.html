Prepare

The Mushop application can run in "mock mode" where service responses are mocked, yet the whole system is fully functional. This is mainly used for quick demos and testing.

Check kubectl context: 
kubectl config current-context
# docker-for-desktop
Check kubectl namespace configuration: 
kubectl config view | grep namespace
